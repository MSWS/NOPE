# NOPE 反作弊模式

一个相当酷的 Minecraft 反作弊。

### 免责声明

NOPE 支持 1.15。X **只是**。 默认情况下，NOPE 将不会在 1.14 上加载 - 服务器。 在
更改插件 yml 时可能解决这个问题，NOPE 可能会在过时版本上产生许多问题。

## 功能

- 广泛的移动检查
- 复杂和彻底的配置能力
- 第三方插件支持 mcMMO 和 AdvancedEnchants
- 自定义封禁动画
- 完全自定义检查
- 前后一致的更新和开发者支持
- 热门封禁插件支持
